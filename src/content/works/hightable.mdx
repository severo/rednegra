HighTable is a React component created by [Kenny Daniel](https://www.linkedin.com/in/kenny-daniel/) for [Hyperparam](/references/hyperparam). It displays large datasets efficiently using virtual scroll. From 2024, I had the opportunity to become co-maintainer of the library, and I contributed to it by adding features like keyboard navigation, sort, rows selection, etc.

<Figure
  id="./hyparam-hightable.png"
  href="https://github.com/hyparam/hightable"
/>

In 2026, I published a blog post (on my [blog](https://rednegra.net/blog/20260212-virtual-scroll/) and on [Hyperparam's blog](https://blog.hyperparam.app/hightable-scrolling-billions-of-rows/)) about the work I did to allow scrolling through billions of rows in a table, while maintaining good performance and accessibility. The feature was added to [HighTable](https://github.com/hyparam/hightable), and it required such amount of research and effort that we decided to write about it in details.
